Assumptions

This calculator provides a theoretical estimate for jet-in-air cell sorters using sheath pressure, nozzle diameter, drop drive frequency, and empirical correction factors. Results should be interpreted as estimates and validated experimentally for critical applications.

The tool is intentionally universal: it does not rely on sorter-specific presets.
